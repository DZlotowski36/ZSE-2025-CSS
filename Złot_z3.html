Właściwości formatowania w:

HTML

CSS
Zmiana wyglądu elementów w HTML jest dość trudna (ze względu na atrybuty i znaczniki rozproszone w różnych częściach kodu, co łączy się ze strukturą dokumentu). W CSS mamy rozdzieloną strukturę i prezentację dokumentu, co pozwala na umieszczenie wszystkich poleceń formatowania w jednym miejscu (tzw. arkuszu stylów), które można powiązać z określonymi elementami HTML. Takie podejście sprawia, że zmiana wyglądu strony jest dużo bardziej efektywna.

Korzyści z używania CSS

  1. Starsze atrybuty i znaczniki HTML odpowiedzialne za formatowanie (przestarzałe elementy) będą stopniowo eliminowane przez przeglądarki na rzecz zalecanych deklaracji CSS.
  2. Dokumenty oparte na CSS są zwykle bardziej przejrzyste i krótsze.
  3. CSS umożliwia łatwe zarządzanie wieloma dokumentami poprzez zastosowanie zewnętrznych arkuszy stylów.
  4. Za pomocą klas selektorów można znacznie skrócić kod, określając formatowanie w jednym miejscu, co później ułatwia zmiany na wielu elementach jednocześnie.
  5. CSS daje możliwość stosowania różnych jednostek i metod definiowania kolorów.
  6. CSS oferuje różne poziomy pogrubienia czcionki (aż 9 opcji).
  7. CSS dostarcza dodatkowe opcje formatowania tekstu.
  8. Efektowne filtry graficzne dostępne w CSS.
  9. CSS pozwala na dostosowanie wyglądu dokumentów w czasie drukowania.
