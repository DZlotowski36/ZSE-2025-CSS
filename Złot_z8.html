Wewnętrzny arkusz stylów

  1. 1. Gdzie jest wstawiany wewnętrzny arkusz stylów?

    Wewnętrzny arkusz stylów jest wstawiany w sekcji <head> dokumentu HTML, pomiędzy znacznikami <style> i </style>. Wszystkie definicje stylów znajdują się wewnątrz tych znaczników.

  2. 2. Składnia teoretyczna (w ramce, konieczne wcięcia)

    <head>
      <style type="text/css">
        selektor1 { cecha: wartość; }
        selektor2 { cecha2: wartość2; }
      </style>
    </head>

  3. 3. Jakie mogą być selektory?

    Selektorami w wewnętrznym arkuszu stylów mogą być różne elementy HTML, takie jak:

    • <p> - akapit
    • <h1> - nagłówek poziomu 1
    • <div> - kontener blokowy
    • <span> - element inline
    • .klasa - selektor klasy
    • #identyfikator - selektor identyfikatora
    Selektory określają, do jakich elementów HTML zastosowane zostaną style.

  4. 4. Jakie mogą być cechy i wartości?

    Cecha to właściwość elementu, którą chcemy zmienić. Przykłady cech to:

    • color - kolor tekstu
    • background-color - kolor tła
    • font-size - rozmiar czcionki
    • margin - margines
    Wartości to konkretne ustawienia dla cech. Przykłady wartości to:
    • red, #FF0000 - dla color
    • 10px, 1em - dla font-size
    • 0, 10px - dla margin
    Cechy i wartości łączy się w deklaracjach stylów wewnętrznego arkusza stylów, aby określić wygląd elementów.
